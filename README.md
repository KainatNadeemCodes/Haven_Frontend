# Haven – Neuro-Inclusive Safety Platform

Haven is a specialized safety and sensory regulation platform designed for neurodivergent individuals. Built with a focus on **WCAG 2.1 AA compliance** and **Cognitive Load Theory**, Haven provides a literal, non-metaphorical interface to assist users in preparing for social situations, managing sensory environments, and accessing social stories.

## 🚀 Key Features

- **Prepare Mode:** AI-assisted (simulated) literal step-by-step planning for new environments.
- **Sensory Regulation:** Custom soundscape builder and "Calm" tab for immediate sensory support.
- **Social Stories:** A library of visual and text-based stories to help navigate social cues.
- **Caregiver Mode:** A secure, PIN-gated dashboard for specialized controls.
- **Neuro-Inclusive UI:** Low-contrast design tokens, reduced motion support, and high-legibility typography.

## 🛠️ Tech Stack

- **Frontend:** React 18 + TypeScript 5
- **Build Tool:** Vite 5
- **Styling:** Tailwind CSS (Semantic HSL design tokens)
- **UI Components:** Radix UI primitives + shadcn/ui
- **State & Logic:** React Query + React Router DOM
